- vytvořte aplikaci, která bude mít dvě tlačítka a každé z nich bude zajišťovat zobrazení jiného fragmentu
- volná inspirace: https://github.com/zizanek/github-tz-pma-2024/tree/main/MyApp008bFragmentsExample1
